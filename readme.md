# Consulte Website
The website for Consulte Comércio de Ambientes.

## Getting started

Install dependencies:

``` bash
$ npm install -g gulp
$ cd consulte
$ npm install
```

Run server:

``` bash
$ gulp serve
```

Run server production:
``` bash
$ gulp serve:dist
```

To make a production build:
``` bash
$ gulp serve:dist
```
